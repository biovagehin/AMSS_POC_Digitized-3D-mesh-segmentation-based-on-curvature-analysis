# Calcul du plan des moindres carré

Ce dossier contient :
- un fichier nuage_de_points.txt contenant les coordonées d'un nuage de points
- un notebook least_squares.ipynb permettant de calculer et d'afficher le plan des moindres carrés de ce nuage de points.

Fait par Biova Géhin dans le cadre du cours 'Techniques et outils pour la preuve de concept' du M2 AMSS.
